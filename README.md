# OpenCV Video Recorder

Python과 OpenCV를 사용하여 구현한 간단한 실시간 비디오 레코더입니다.  
웹캠 영상을 화면에 출력하고, 사용자의 입력에 따라 영상을 녹화하여 파일로 저장할 수 있습니다.

---

## 주요 기능

### 1. 카메라 영상 출력
- OpenCV의 `VideoCapture`를 사용하여 웹캠 영상을 실시간으로 화면에 표시합니다.

### 2. 비디오 녹화
- OpenCV의 `VideoWriter`를 이용하여 카메라 영상을 동영상 파일(`output.avi`)로 저장합니다.
- MJPG 코덱을 사용하여 영상을 인코딩합니다.

### 3. Preview / Record 모드
- **Preview 모드**: 카메라 영상만 화면에 표시
- **Record 모드**: 카메라 영상을 녹화하여 파일로 저장
- 녹화 중에는 화면에 **Recording 텍스트와 빨간 원 표시**가 나타납니다.

### 4. FPS 조절
- 사용자가 키 입력을 통해 녹화 FPS를 증가 또는 감소시킬 수 있습니다.

### 5. 화면 좌우 반전
- 카메라 영상을 좌우 반전하여 거울처럼 사용할 수 있습니다.

---

## 키보드 조작

| 키 | 기능 |
|---|---|
| Space | 녹화 시작 / 종료 |
| F | 화면 좌우 반전 |
| + | FPS 증가 |
| - | FPS 감소 |
| ESC | 프로그램 종료 |
